# UI Assets

В этой папке находятся изоюражения, используемые ботом в Telegram.
Для правильной работы бота поместите сюда свои изображения с точно такими же названиями:

- `configs.png`
- `menu.png`
- `mtproto.png`
- `payment.png`
- `profile.png`
- `settings.png`

*Примечание: при острой необходимости вы можете изменить эти имена файлов в файле `core/ui.py`.*
